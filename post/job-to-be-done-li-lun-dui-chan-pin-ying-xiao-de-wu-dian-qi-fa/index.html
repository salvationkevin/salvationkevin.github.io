JTBD理论给“产品营销”带来哪些启发？

2022-04-03 5 min read # 观点 # JTBD # 产品营销

何为JTBD（job-to-be-done）？

你可能听说过JTBD，但并不完全理解它是什么。本质上来说，JTBD理论旨在解释人们为什么做出购买（或不购买）行为。

对marketers来说，产品营销是将产品引入市场的过程。人们买或不买产品都是marketers所特别关心的事儿，JTBD理论既适用于B2B市场，也适用于B2C市场。

关于job-to-be-done理论的五点认识

认识一：我们每个人都有待办任务要完成

人的一生充满各种“艰难时刻”（struggling moment）。我们借由产品或服务帮助解决所面临的种种困境。

譬如，当你感觉无聊时，你或许可以通过以下方式得到缓解：

  • 刷抖音，或微信朋友圈
  • 听音乐
  • 看书
  • …

又如，当你感觉到工作压力大时，你或许可通过以下方式得以缓解：

  • 通过软件协助项目管理
  • 通过纸质笔记本记录下心情
  • 向公司请假，完成一次海滨度假之旅调整下
  • …

以上就是我们每个人在感知到“艰难时刻”时，可通过诸多方式（产品或服务）解决问题。

认识二：可感知到的进步诉求（perceived progress）

譬如，当你感觉到工作压力大时，要如何缓解呢？可能存在以下两个选择：

  • 第一种：通过办公软件的协助，做好项目管理，缓解工作压力
  • 第二种：干脆向公司提出休假申请，在海滩边给自己放松减压；

但在尝试穷尽这种选择项时，其中的关键问题在于：如果没有有关工作压力大的上下文情境，那么这两种选择也仅仅是选择项而已。即使您可以列出很多选项，最终都对解决工作压力大这个问题没有针对性的有效方案，因为缺乏了具体情境内容，我们无法了解任何选项背后的进步诉求是什么，从而也就无法提供真正有效的解决方案。

认识三：诱因事件

要想要对情境内容有所了解，就需要我们针对诱因事件和期望结果做更深入的挖掘。我们每个人每天都在做一些决策，无论是买或不买什么东西。

举例：在工作中感受到压力？正如很多人一样，都会有如此感受。但如果说对John来说，他正在某个项目上，而且已经加班加点好几个月的时间了，这时候经理对他说，你要是在下个项目上表现突出的话，会得到晋升。John似乎感受到了激励，但回头当他在凌晨1点走出办公室门口时，突然意识到这样的工作节奏已经持续了快四个月了，不禁心头一阵感慨掠过。经理的口头允诺似乎变得遥遥无期。或许正是在个时候，John决定申请去海边度假一段时间调整自己的工作状态。

认识四：始终与过程有关

以上图为例，从工作压力大的状态下到完全放松的状态，你的产品或服务可在某个恰当的时机出现。譬如，在做功课阶段，你的产品可能是与搜索引擎（比如Google）或朋友口碑推荐来竞争，当到预定住宿时，你提供的产品或服务可能是与Airbnb，或Booking这样的OTA软件进行竞争。

在这个完整的旅程中，你不需要提供从头到尾的产品或服务，而是从某个切入点进入，只是旅程的一部分，为用户创造价值。

认识五：围绕进步展开设计

进步（progress）一词似乎听上去一头雾水，无法确定它的具体所指。其实，这里讲的就是有关用户的待办任务，一旦他/她完成该任务就能收获进步感受，因为涉及其中的产品或服务帮助他/她克服了某些艰难时刻。

举例：某个用户的待办任务是“帮助他/她实现工作日任何情况下都能够准时上班”

该用户可能想到“雇佣”的产品有：
a.闹钟 b.床 c.汽车

那么，围绕这些雇佣产品，可能的重要特点设计体现为：
a.闹钟（声音大） b.床（舒适，整夜安心睡眠） c.汽车（加速快；第三方车辆服务）

对产品营销的五点启发

  1. 在做用户研究时，要寻找用户的待办任务；
  2. 在做广告活动时，凸显出用户所感知到的进步；
  3. 在做精准营销时，需要寻找心理因素或行为诱因；
  4. 在做产品设计时，要在整个过程中找准时机停下和开始设计产品；
  5. 在做广告文案时，要凸显出那些对完成待办任务来说最重要的特点；
